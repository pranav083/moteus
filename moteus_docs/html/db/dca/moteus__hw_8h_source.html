94 // The "FIRMWARE_VERSION" is a misnomer. It instead is the equivalent

96 // change in a way that would not result in equivalent behavior across
97 // an upgrade/downgrade.

99 // Version history:
100 
101 // # 0x0101 #
102 //
103 // * Fixed the calculation of feedforward voltage to have the correct
104 // sign for the velocity component. Previous firmwares,
105 // inappropriately applied a negative feedforward term, which
106 // counteracted rotation instead of being an actual feedforward.
107 
108 // # 0x0102 #
109 //
110 // * Removed servo.feedforward_scale entirely
111 
112 // # 0x0103 #
113 //
114 // * Added servo.pwm_scale, and for r4.8 boards changed the default
115 // value of pwm_comp_off / pwm_comp_mag.
116 
117 // # 0x0104 #
118 //
119 // * Added configurable PWM rates, which changed default values of
120 // pwm_comp_mag and pwm_comp_off.
121 
122 // # 0x0105 #
123 //
124 // * Switched to a new encoder and position subsystem.
